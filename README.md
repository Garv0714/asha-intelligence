# 🤝 Asha Intelligence

**AI-Powered Humanitarian Nutrition Platform**

Predictive social intelligence platform that identifies and protects children from hunger — before crisis becomes irreversible.

## 🌟 Vision

Every child deserves to learn on a full stomach. Asha Intelligence uses AI-driven analytics to empower schools, NGOs, and governments with actionable nutrition insights across every district.

## ✨ Features

- **🧠 AI Risk Prediction** — ML-powered hunger risk scoring for every school and district
- **🗺️ Hunger Risk Maps** — Geospatial visualization of nutritional vulnerability zones
- **📈 Predictive Analytics** — 7 to 90-day hunger forecasting with 89% accuracy
- **🏫 School Intelligence** — Attendance, meal participation, and dropout correlation tracking
- **🛡️ Intervention Engine** — AI-recommended actions with projected impact scoring
- **⚡ NGO Resource Simulator** — Optimal resource allocation with budget-aware impact estimation
- **🏆 District Severity Ranking** — Comparative hunger severity, urgency, and educational impact
- **🎤 Presentation Mode** — Fullscreen storytelling flow optimized for stakeholder demos

## 🛠️ Tech Stack

- **Framework:** Next.js 16 (Turbopack)
- **Styling:** Tailwind CSS v3 with custom humanitarian design system
- **Animations:** Framer Motion
- **Charts:** Recharts
- **Maps:** React-Leaflet with CartoDB light tiles
- **Fonts:** Inter (Google Fonts)

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the platform.

## 📁 Architecture

```
src/
├── app/              # Next.js App Router pages
│   ├── dashboard/    # Dashboard with sidebar layout
│   │   ├── analytics/    # Hunger map + trends
│   │   ├── districts/    # District list + detail
│   │   ├── interventions/# AI recommendations + tracking
│   │   ├── presentation/ # Hackathon presentation mode
│   │   ├── reports/      # AI-generated reports
│   │   ├── schools/      # School list + onboarding
│   │   ├── severity/     # District severity rankings
│   │   ├── settings/     # Platform settings
│   │   └── simulator/    # NGO resource allocation
│   └── page.tsx      # Landing page
├── components/       # Reusable UI components
├── data/synthetic/   # Synthetic datasets
├── lib/              # Utilities
└── types/            # TypeScript definitions
```
